# Athlete 送迎ルート 使い方マニュアル

## 1. ログイン

1. 画面上部のログイン欄で利用者を選びます。
2. PINを入力して「ログイン」を押します。
3. ログイン後に「今日の送迎」「地図・配車表」「利用者登録・CSV」を操作できます。

## 2. 今日の送迎を作る基本手順

1. 「今日の送迎」を開きます。
2. 「対象日」が正しい日付になっているか確認します。
3. 「曜日予定から乗る方を選ぶ」に、その曜日に来る予定の方が五十音順で表示されます。
4. 本日来ない方は「本日来ない」にチェックします。
5. 早帰り、迎え時間、送り時間、配車表の備考がある場合は同じ行で修正します。
6. 乗せる方だけ「この方を追加」を押します。
7. 全員をまとめて入れたい場合だけ「表示中の搭乗者をまとめて送迎に反映」を押します。

## 3. 予定外・振替の方を追加する

1. 「予定外・振替で来る利用者」のプルダウンを開きます。
2. 登録済み利用者から対象者を選びます。
3. 「この方を一覧に追加」を押します。
4. 一覧に表示されたら、必要に応じて「この方を追加」を押して今日の送迎へ入れます。

## 4. 車両の考え方

- 1号車、2号車は車いす対応です。
- 3号車、4号車は普通席の利用者向けです。
- 車いす利用者を追加すると、個別の配車車両は1号車または2号車へ自動で割り当てます。
- 画面上の「車両選択」は、新しく追加する方の標準車両です。今日の送迎に入った後は、利用者ごとの「配車車両」で変更できます。
- 定員を超えた場合は、ルート作成前に確認メッセージが表示されます。

## 5. ルート作成と配車表出力

1. 今日の送迎に必要な方を追加します。
2. 必要に応じて「車両を自動で分ける」を押します。
3. 「ルートを作る」を押します。
4. 「地図・配車表」画面に移動し、送迎順と地図を確認します。
5. ドライバーへ渡す場合は、配車表の「コピー」「CSV」「印刷」を使います。

## 6. 利用者登録・CSV

1. 「利用者登録・CSV」を開きます。
2. 1人ずつ登録する場合は、名前、住所、人数、車いす、曜日予定、備考を入力して登録します。
3. 一括登録する場合は、CSVまたはExcelファイルを選びます。
4. 取り込み内容を確認して登録すると、今日の送迎のプルダウンや曜日予定に反映されます。

## 7. よくある注意点

- 「追加済み」と表示されている方は、すでに今日の送迎に入っています。
- 「本日来ない」にチェックした方は、「この方を追加」が押せません。
- 日付を変えた場合は、その日付の曜日予定に切り替わります。
- 住所が正しく見つからない場合は、川崎市川崎区から始まる住所に直して再度ルートを作成してください。
