# 川崎区 介護送迎ルート

介護サービス向けの送迎ルート作成Webアプリです。既存のFM大師業務アプリとは別ページとして `care-route/` に配置しています。

## 主な機能

- 川崎市川崎区内の送迎を想定
- 1号車、2号車、3号車、4号車の定員を反映
- 2号車は車椅子対応
- 患者様の住所、人数、車椅子有無、迎え時間の制約を登録
- 出発地、到着地、出発予定時刻を指定
- 時間制約を考慮した送迎順の自動作成
- OpenStreetMap/OSRMの道路ルートを地図に表示
- 走行距離、所要時間、到着目安、注意点を表示
- 現在地を出発地に設定
- 現在時刻で再計算
- 3分ごとのルート自動更新
- 前回確認からの所要時間差分を表示
- Googleマップで開いて端末側のライブ交通状況を確認

## 車両マスター

- 1号車: 座席5名、車いす2台対応
- 2号車: 座席5名、車いす2台対応
- 3号車: 定員8名
- 4号車: 定員5名

## 注意

現在は納品前の最終確認用Webアプリです。現行構成では、OSRMの道路ルート再取得、現在時刻計算、時間帯補正、Googleマップ確認リンクで可能な範囲までリアルタイムに近づけています。

本番運用で一方通行、交通規制、渋滞、施設名検索の精度を高める場合は、Google Maps Platform、Mapbox、ゼンリン系APIなどの商用地図APIを接続してください。
