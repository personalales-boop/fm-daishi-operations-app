# 川崎区 介護送迎ルート

介護サービス向けの送迎ルート作成Webアプリです。既存のFM大師業務アプリとは別ページとして `care-route/` に配置しています。

## 主な機能

- 川崎市川崎区内の送迎を想定
- 1号車、2号車、3号車、4号車の定員を反映
- 2号車は車椅子対応
- 患者様の住所、人数、車椅子有無、迎え時間の制約を登録
- 出発地、到着地、出発予定時刻を指定
- 時間制約を考慮した送迎順の自動作成
- OpenStreetMap/OSRMの道路ルートを地図に表示
- 走行距離、所要時間、到着目安、注意点を表示

## 車両マスター

- 1号車: 定員4名
- 2号車: 定員4名、車椅子対応
- 3号車: 定員8名
- 4号車: 定員5名

## 注意

現在は納品確認用のWebアプリです。リアルタイム渋滞は未接続です。

本番運用で一方通行、交通規制、渋滞、施設名検索の精度を高める場合は、Google Maps Platform、Mapbox、ゼンリン系APIなどの商用地図APIを接続してください。
